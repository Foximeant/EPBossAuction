EPBossAuction - Руководство пользователя
Описание
Аддон для проведения аукционов в рейде по системе EPGP. Позволяет игрокам делать ставки EP на предметы с боссов, а Loot Master'у отслеживать и управлять ставками.

Установка
Распакуйте архив с аддоном в папку Interface\AddOns\
Убедитесь, что папка называется EPBossAuction
Запустите или перезапустите World of Warcraft
В списке аддонов убедитесь, что EPBossAuction отмечен галочкой

Команды
/epba - открыть/закрыть главное окно аддона
/epgpfind - диагностика EPGP аддона (поиск и тестирование)
/epbasave - принудительное сохранение данных
/epbaupdate - принудительное обновление вашего EP

Основной функционал
Для всех игроков
Выбор босса - из выпадающего списка выберите нужного босса
Выбор предмета - кликните по строке с предметом в таблице или выберите из выпадающего списка
Ввод ставки - введите число EP в поле ввода (минимум 1000 EP, 0 - отмена ставки)
Отправка ставки - нажмите кнопку "Сделать ставку" или клавишу Enter

Для Loot Master'а
Автоматическое определение - аддон определяет, являетесь ли вы лутермастером
Прием ставок - все ставки от игроков автоматически обрабатываются
Синхронизация - данные автоматически рассылаются всем игрокам в рейде
Завершение аукциона - кнопка "Очистить таблицу" удаляет все ставки для выбранного босса
Отправка в чат - кнопка "В чат" выводит текущие ставки в рейд
Интерфейс
Таблица предметов - отображает все предметы выбранного босса
Топ-2 ставки - для каждого предмета показываются две лучшие ставки
Цветовая индикация - ники игроков отображаются цветом их класса
Подсветка - выбранный предмет выделяется голубым фоном
Тултипы - при наведении на иконку или название показывается информация о предмете
Ваш EP - отображается текущее количество ваших очков EP
Красная подсветка - при вводе ставки больше вашего EP, текст становится красным

Система ставок
Минимальная ставка: 1000 EP
Ставка 0 означает отказ от текущей ставки
Ставка должна быть больше текущей максимальной ставки на предмет
Нельзя поставить больше, чем у вас есть EP

Синхронизация данных
При входе в мир данные автоматически загружаются
Лутермастер автоматически рассылает данные всем в рейд
Новые игроки автоматически запрашивают данные при входе в рейд
Автосохранение каждые 10 секунд
Данные сохраняются при выходе из игры
ElvUI поддержка
При наличии ElvUI аддон автоматически применяет стилизацию под ElvUI.

Примечания
Аддон работает только в группе или рейде при системе распределения добычи с ответственным
Для работы требуется EPGP аддон (EPGP, EPGP_Auction, CEPGP или EPGPCore)
Все данные синхронизируются между игроками через аддон-каналы
