# EP Boss Auction

[![EPBA version](https://img.shields.io/badge/version-1.4.1-green.svg)](https://github.com/Foximeant/EPBossAuction)
[![WoW 3.3.5](https://img.shields.io/badge/WoW-3.3.5-blue.svg)]()

**EP Boss Auction** – аддон для World of Warcraft (Sirus 3.3.5+), предназначенный для проведения аукционов на основе EPGP прямо в рейде. Позволяет игрокам делать ставки EP на предметы с боссов, а лутеру – управлять процессом, просматривать текущие ставки и синхронизировать данные с рейдом.

## Особенности

- 📋 **Таблица предметов** по каждому боссу с отображением топ-N ставок (настраивается).
- 🎨 **Настраиваемый внешний вид**: размер шрифта, ширина колонок, высота строки, цвета фона, цвет названий (золотой или по редкости предмета).
- 🔒 **Блокировка ставок** – лутер может временно запретить приём ставок, состояние видно всем.
- 🖱️ **Кнопка у миникарты** – перетаскиваемая, с сохранением позиции, отображает статус лутера.
- ⚙️ **Полноценное окно настроек** (4 вкладки) – гибкая кастомизация под себя.
- 📡 **Синхронизация данных** между лутером и участниками рейда через аддон-канал.
- 🔗 **Интеграция с EPGP** – автоматическое получение EP игрока, учёт твинков.
- 💬 **Вывод текущих ставок в рейд-чат** (одной кнопкой).
- 🎨 **Поддержка ElvUI** – основное окно и кнопки стилизуются под ElvUI (опционально, чекбокс блокировки остаётся стандартным во избежание багов).

## Требования

- World of Warcraft 3.3.5 (Wrath of the Lich King)
- Установленный EPGP-аддон (любой из: EPGP, EPGP_Auction, CEPGP, EPGPCore)
- (Опционально) ElvUI для скина

## Установка

1. Скачайте последнюю версию из [релизов](https://github.com/Foximeant/EPBossAuction/releases).
2. Распакуйте архив в папку `World of Warcraft\Interface\AddOns\`.
3. Убедитесь, что папка называется `EPBossAuction`.
4. Запустите игру и включите аддон в списке аддонов.

## Использование

### Основные команды

| Команда          | Действие                                     |
|------------------|----------------------------------------------|
| `/epba`          | Открыть/закрыть главное окно                |
| `/epbaoptions`   | Открыть окно настроек                       |
| `/epbaupdate`    | Принудительно обновить свой EP              |
| `/epbasave`      | Принудительно сохранить данные               |
| `/epgpfind`      | Поиск EPGP-аддона (для отладки)             |
| `/epbazoom+ /- /reset` | Увеличить/уменьшить/сбросить масштаб окна |

### Интерфейс

- **Выбор босса** – выпадающий список слева вверху.
- **Блокировка ставок** – чекбокс справа от выпадающего списка (доступен только лутеру). При включении ставки не принимаются, кнопка «Сделать ставку» блокируется у всех.
- **Таблица предметов** – для выбранного босса отображается список предметов. Названия предметов окрашены в золотой или по редкости (настраивается). Во второй колонке – топ-2 ставки (количество настраивается). Клик по строке выбирает предмет для ставки.
- **Поле ввода ставки** – введите сумму (целое число) и нажмите Enter или кнопку «Сделать ставку». Если сумма превышает ваш EP, текст становится красным.
- **Кнопка «Запросить»** – для не-лутера: запросить актуальные данные у лутера.
- **Кнопка «Очистить таблицу»** – (только лутер) завершает аукцион, удаляя все ставки для текущего босса.
- **Кнопка «В чат»** – (только лутер) выводит текущие ставки в рейд-чат.

### Кнопка миникарты

- Круглая иконка у миникарты (перетаскиваемая). Клик – открыть/закрыть главное окно.
- При наведении показывает статус лутера.
- Позиция и видимость настраиваются.

## Настройки

Окно настроек (вызывается через `/epbaoptions` или кнопку ⚙ в главном окне) содержит четыре вкладки:

### Общие
- Режим отладки (вывод в чат отладочной информации).
- Минимальная ставка (фиксирована, задаётся в коде).
- Автоматически запрашивать данные при входе в рейд.
- Подтверждать ставки (диалог подтверждения).
- Включить звук при ставке (файл задаётся в коде).

### Таблица
- Размер шрифта названий предметов и ставок.
- Ширина колонок.
- Количество отображаемых ставок (1–5).
- Цвета строк (чётные, нечётные, выделенная, при наведении).
- Режим цвета названий предметов (золотой / по редкости).
- **Высота строки** (пиксели, иконка предмета масштабируется под неё).
- **Привязка тултипа** при наведении на строку (у курсора, справа, слева, сверху, снизу и т.д.).

### Миникарта
- Показывать кнопку у миникарты.
- Размер кнопки (20–40 пикс).
- Радиус от центра (50–100 пикс).
- Кнопка «Сбросить позицию» (возвращает на 3 часа).

### Окно
- Прозрачность окна (30–100%).
- Масштаб окна (кнопки +, -, 100%).
- Блокировка окна (запретить перемещение).
- Кнопка «Сбросить позицию окна» (в центр экрана).
